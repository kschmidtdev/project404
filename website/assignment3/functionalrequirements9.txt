<p><strong>Name:</strong> Saving </p><p><strong>Description:</strong> Once the battle has ended, the user must save from the overmap </p><p><strong>Actors:</strong> ALL </p><p><strong>Use Case:</strong> </p><h3></h3><p><i>Normal:</i> </p><p>1. Menu: Access the menu by pressing the menu button </p><p>2. Selection: Select the save option </p><p>3. Selection: In the save menu select an empty space. </p><p><i>Abnormal 1:</i> </p><p>1. Menu: Access the menu by pressing the menu button </p><p>2. Selection: Select the save option </p><p>3. Selection: Select already in use save slot </p><p>4. Input: Answer with button on Yes overwrite or No do not. </p><p><i>Abnormal 2:</i> </p><p>1. Menu. Access the menu by pressing the menu button </p><p>2. Selection: Select improper action. </p><p>3. Selection: Choose cancel and go back to main menu. </p><p>4. Selection: Select proper option. </p><p>5. Selection: In the save menu select an empty space. </p><h3></h3><p><strong>Features Used:</strong> </p><ul><li>Selection Cursor </li><li>Action button </li><li>World Map Menu </li></ul><p><strong>Exceptions:</strong> </p><ul><li>Selecting improper button (simply go back and choose correct) </li><li>Overwriting files (ask if they are sure) </li></ul><p><strong>Actor:</strong> </p><p><i>Max:</i> Max knows and utilizes saving as a given. </p><p><i>Hubert:</i> Hubert will know to save, but he may need to learn how to reach the save menus so as to use them. </p>

<p><strong>Name:</strong> Quitting </p><p><strong>Description:</strong> The user wishes to exit the game from the main map </p><p><strong>Actors:</strong> ALL </p><p><strong>Use Case:</strong> </p><h3></h3><p><i>Normal:</i> </p><p>1. Menu: Press menu button to access the menu. </p><p>2. Selection: Select the quit option. </p><p>3. Selection: Select Yes when asked if they are sure. </p><p><i>Abnormal:</i> </p><p>1. Menu: Press menu button to access the menu. </p><p>2. Selection: Select the quit option. </p><p>3. Mistake: Select No to go back to game. </p><h3></h3><p><strong>Features Used:</strong> </p><ul><li>Selection Cursor </li><li>Action button </li><li>World Map menu </li></ul><p><strong>Exceptions:</strong> </p><ul><li>Selecting No, back to map </li></ul><p><strong>Actor:</strong> </p><p><i>All:</i> </p><p>Quitting is intuitive and imperative in operation for both users. Since it is easily identifiable on the menus, it should be simple to apply. </p>
