<b>Guidelines</b>
<ul class="typicallist"><li>The software will be written in C++ </li><li>The renderer, and input will use the SDL (<a href="http://www.libsdl.org" rel="nofollow">http://www.libsdl.org</a>) </li><li>Sound will be implemented using SDL_mixer (<a href="http://www.libsdl.org/projects/SDL_mixer/" rel="nofollow">http://www.libsdl.org/projects/SDL_mixer/</a>)
</li>
<li>Font loading/rendering will be implemented using SDL_ttf (<a href="http://www.libsdl.org/projects/SDL_ttf/" rel="nofollow">http://www.libsdl.org/projects/SDL_ttf/</a>) </li>
<li>XML loading/saving will be implemented using tinyXML (<a href="http://www.grinninglizard.com/tinyxml/" rel="nofollow">http://www.grinninglizard.com/tinyxml/</a>) </li>
<li>Unit tests will be implemented using cxxtest (<a href="http://cxxtest.sourceforge.net/" rel="nofollow">http://cxxtest.sourceforge.net/</a>) </li>
<li>User profiles will be encrypted for protection </li><li>Game archives (data and database, if they are separate) will be encrypted for protection and integrity assurance </li><li>Our game will not be a clone or copy of the gameplay found in another game title </li><li>Our game will not include or use media or trademarks not legally licensed to us </li><li>The game will only utilize the 8 directions and 5 keys we have chosen (Start, Select, Cancel, Confirm and Menu) </li></ul>
