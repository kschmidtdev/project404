<p><strong>Name:</strong> Playing Tutorial </p><p><strong>Description:</strong> The user must start the tutorial mode </p><p><strong>Actors:</strong> Hubert </p><p><strong>Use Case:</strong> </p><h3></h3><p>1. Selection: Move the cursor to “Tutorial?</p><p>2. Start: Use enter to start the tutorial. </p><p><strong>Features Used:</strong> </p><ul><li>Selection Cursor </li><li>Menu </li><li>Button </li></ul><p><strong>Exceptions:</strong> </p><blockquote>N/A 

</blockquote><p><strong>Actor:</strong> </p><p><i>Max:</i> Max doesn&#x27;t despise tutorials, but since he is familiar with the tactics games, he feels that utilizing a boring, long tutorial will not benefit him. Frankly it prevents him from getting his characters up to snuff fast enough. He would much rather get started right away, so he choose to not do the tutorial </p><p><i>Hubert:</i> Hubert isn&#x27;t exactly sure why he is required to use a profile, but since he wants to get to the game, he is willing to get one started. At least he&#x27;ll know he isn&#x27;t spending too much time on games. </p>

<p><strong>Name:</strong> Load Game </p><p><strong>Description:</strong> The user selects their saved game </p><p><strong>Actors:</strong> ALL </p><p><strong>Use Case:</strong> </p><h3></h3><p><i>Normal:</i> </p><p>1. Selection: Move the cursor to Load game and Enter </p><p>2. Selection: Select preferred saved game. </p><p>3. Validate: User accepts when asked if this is the game they wish to load </p><p><i>Abnormal 1:</i> </p><p>1. Selection: Move the cursor to Load game and Enter </p><p>2. Selection: Select preferred saved game. </p><p>3. Validate: User denies when asked if this is the game they wish to load </p><p>4. Selection: User is returned to load menu, and may select again. </p><p><strong>Features Used:</strong> </p><ul><li>Selection Cursor </li><li>Menu </li><li>Button </li><li>Loading </li></ul><p><strong>Exceptions:</strong> </p><ul><li>Incorrect save game, allow user to go back before finalizing choice </li></ul><p><strong>Actor:</strong> </p><p><i>All:</i> They all know that to load a game is imperative to playing the game. </p>
